# gome_intrinsic_height_memo

Efficient height caching widget to reduce layout passes in complex Flutter UIs.

Useful for:

- SliverLists with dynamic height
- Expandable panels
- Lazy-loaded lists

## Usage

```dart
IntrinsicHeightMemo(
  child: YourComplexWidget(),
)
```
